목표

반도체 PCB 기판의 경계성 불량의 넓이를 측정하여 양불 판정 모델을 개발하는 것이 목표였습니다.

정의

PCB 기판에서 발생하는 불량 중 돌기와 슬릿을 경계성 불량으로 지칭합니다. PCB 기판에서는 에폭시 위에 구리로 모양을 본뜨게 되는데, 구리가 에폭시로 더 많이 형성되어 예상치 못한 곳으로 전류가 흐르게 되는 경우를 돌기라고 지칭하고 더 적게 형성되어 전류가 흘러야하는데 흐르지 못하는 경우를 슬릿이라고 지칭합니다.

접근 방법

1. 돌기와 슬릿의 넓이를 측정하려면 돌기와 슬릿 부분을 분할하여야합니다. 이를 위해 이진화 알고리즘을 통해서 에폭시와 구리 영역을 분할합니다.

2. Object Detection 모델을 통해 돌기와 슬릿이 일어난 부분을 검출합니다.

3. 검출된 bbox의 전체 영역과 돌기와 슬릿이 발생된 영역을 계산하고 그 둘의 비율을 계산합니다. 그리고 돌기와 슬릿의 넓이로 정의합니다.

4. 계산된 돌기와 슬릿의 넓이가 특정 임계값을 넘으면 불량, 아니면 정상이라고 최종적으로 판단합니다.

진행 중 발생한 문제점

추가적으로 제공받은 데이터셋에서는 PCB 기판에서 돌기와 슬릿이 대각선 방향으로 존재하였습니다. 따라서 오직 가로, 세로 방향으로 예측하는 기존 Object Detection 모델로는 검출되지 않는 문제가 발생하였습니다.

해결 방법

위의 문제를 해결하기 위해 두 가지 방법을 생각해보았습니다.

1. 푸리에 변환을 통해 가장 강한 직선 성분을 찾고 그 성분과 수직, 수평 방향과의 각도를 계산하여 그 각도의 반대 방향으로 이미지를 회전

2. 기존 Object Detection 모델을 대각선 방향의 물체도 검출할 수 있는 Oriented Bounding Box Detection 모델로 교체

결론

첫 번째 방법의 경우 회전 각도를 부정확하게 찾아 bbox를 검출하지 못하는 경우가 있다는 것을 발견하였습니다.

두 번째 방법의 경우 학습이 잘 진행되고 있다는 것을 발견하였고, 다양한 상황에서 발생하는 이미지로 학습하면 성능을 향상시킬 수 있을 것이라고 판단하였습니다.

따라서, 두 번째 방법을 최종 모델로 결정하여 문제를 해결하였고, 이를 통해 실제 계약까지 성사시킬 수 있었습니다.

진행 중 발생한 문제점

빨간색 박스를 없애기 위해 OpenCV의 inpaint 함수를 사용하였는데, 완벽하게 지워지지 않는 것을 발견하였습니다. 이미지를 확대해보니 단일한 색의 빨간색(255, 0, 0)이 아니라 명도와 채도가 변하면서 빨간색이 칠해져있다는 것을 발견하였습니다. 따라서 범위를 지정하는 OpenCV의 inpaint 함수가 제대로 동작하지 못한다는 결론을 얻을 수 있었습니다.

해결 방법

기존 사용 방법인 OpenCV의 inpaint 함수를 Generation 방법을 사용하는 Inpainting 모델로 교체하였습니다.

결론

기존의 문제였던 빨간색 박스를 성공적으로 제거하여 일반화 성능이 높은 Object Detection 모델을 개발할 수 있었고, 업체측으로부터 좋은 평가를 받을 수 있었습니다.

Abstraction: 해수면 온도는 수치적 기후 예측모델과 해양 예측 모델의 가장 중요한 변수로 전세계 해양의 열 분배, 해양-대기 순환을 결정한다. 해수면 온도 변화는 전세계 적인 기후 변화와 해양 시스템 간의 상호작용으로 인해 발생한다. 최근 다양한 외력요인들이 한반도의 해수면 온도를 높이고 있으며 이로 인해 연안 어로, 양 식업 등 수산업에 막대한 피해를 끼치고 있다. 이러한 해수면 온도의 고수온 현상을 사전에 예측하여 대비한다면 고수온 현상으로 인해 발생되는 피해를 줄 일 수 있을 것이다. 본 연구에서는 해수면 온도의 고수온과 같은 극값(Extreme Value)을 예측하는데 뛰어난 성능을 보이는 수용 장 블록(Receptive Field Block) 기반의 선형사영 인코딩 모듈 LPRFBM(Linear Projected Receptive Field Block Module)을 제안한다. LPRFBM은 시계열 데이터를 시계열 변수 축으로 선형사 영시켜 시간적 요소 뿐 아니라 시계열 데이터의 변수까지도 고려할 수 있다. LPRFBM은 어느 시계열 예측모델의 인코딩에서도 적용할 수 있기 때문에 확장 성이 높고 시간 정보를 부여하는 작업을 따로 해주지 않아도 되기 때문에 편리 하게 사용할 수 있다. 여러 실험의 양적(Quantitative), 질적(Qualtitative)지표를 바탕으로 보았을 때, LPRFBM은 시계열 데이터의 극값 예측성능을 향상시킬 수 있다고 볼 수 있다.
