반도체 PCB기판 양/불 판정을 수행하는 AI Solution 설계

Introduction

기존 방식에 대한 설명은 아래와 같습니다.

1. 조립이 완성된 Package Substrate PCB기판을 AVI(Auto Vision Inspection)장비에 투입시켜 불량 부위의 대략적인 위치를 촬영하여 이미지를 획득

2. 획득한 이미지를 IVS(Industrial Vision System)에 투입시켜 사람이 수동으로 PCB기판의 양/불 판정을 진행하게됨.

3. 2.과정에서 양/불 판정이 어려운 경우, 정밀한 시각적 관찰이 가능한 VRS 장비를 통해 최종 양/불 판정을 하게됨.

기존 방식의 경우, IVS, VRS 장비를 활용하여 검사하는 과정에 사람이 투입되었습니다. 본 프로젝트에서는 이 과정을 AI Solution으로 대체하는 것을 목표로 하였습니다.

Work Flow

설계한 AI Model의 Work Flow는 위의 그림과 같습니다. 먼저, 이미지가 들어오면 각각의 불량 유형으로 분류를 한 후, 해당 불량 유형의 불량 부위의 크기를 재서 고객사의 Spec에 따라 양/불 판정을 진행합니다.

Contribution

제품의 양/불 판정을 위해서는 불량 부위의 크기를 재는 것(위의 그림에서 2. 과정)이 필요한데, 불량 부위의 크기를 잴 수 있도록 불량 부위를 Segmentation하는 Algorithm 개발에 참여하였습니다.

AI Model

SOLO Algorithm의 경우, 물체를 직접적으로 분할하기 때문에 다른 알고리즘들과 비교하여 큰 물체와 작은 물체의 세밀한 부분까지 Segmentation하는 모습을 보여주었습니다. 프로젝트에서 Segmentation하고자 하는 PCB 기판의 경우, 큰 크기의 불량과 작은 크기의 불량 모두 세밀하게 Segmentation을 수행하는 것이 필요하였기 때문에 SOLO Algorithm이 적절하다고 판단하였습니다.

Challenging Point

크게 분할되어야하는 Object들이 더 작은 Object들로 분할되는 경향이 있었습니다. 이는 큰 크기의 Object들보다 작은 크기의 Object들이 더 많이 존재하여, 모델이 학습될 때 가중치가 작은 Object들에 편중되기 때문에 나타나는 현상이라고 파악을 하였습니다.

How To Solve

1. 특정 클래스의 데이터 수가 많아 가중치가 편중되어 학습되는 것을 해결하기 위해 제안된 Seesaw loss를 사용하였습니다.

2. False Positive를 줄이기 위해 여러 Threshold를 학습 과정에 사용한 Cascade Mask RCNN을 사용하였습니다.
